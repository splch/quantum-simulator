## Introduction

A simple quantum simulator implemented in Go.

## Requirements

- Go 1.21 or higher

## Install

To install the library:

```shell
go get github.com/splch/quantum_simulator
```

## Setup

To download dependencies:

```shell
go mod tidy
```

## Usage

To `build` the project:

```shell
make build
```

To `test` the library:

```shell
make test
```

To `run` the simulator:

```shell
make run
```
