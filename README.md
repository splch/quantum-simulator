## Introduction

A simple quantum simulator implemented in Go.

## Requirements

- Go 1.21 or higher

## Usage

To `get`:

```shell
go get github.com/splch/quantum_simulator
```

To `run`:

```shell
go run main.go
```

To `test`:

```shell
go test ./pkg/quantum_simulator/... -v
```
